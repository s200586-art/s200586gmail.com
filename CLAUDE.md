Для КАЖДОГО счётчика сделай 3 запроса:
- a) KPI за вчера: `/stat/v1/data?ids={id}&metrics={metrics}&date1=yesterday&date2=yesterday`
- b) KPI за позавчера: `/stat/v1/data?ids={id}&metrics={metrics}&date1=2daysAgo&date2=2daysAgo`

## Шаг 2: Анализируй как SEO-специалист

- Если >70% Direct — SEO не работает
- Если Referral доминирует на субдомене — органики нет, люди приходят только с главного сайта
- Если мессенджер = 0 несколько дней подряд — кнопка сломана
- Если главная страница >80% просмотров — каталог не продвигается
- Если 0 звонков при >50 пользователях — кнопка телефона не работает
